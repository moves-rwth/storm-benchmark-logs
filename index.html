Setup

The benchmarks were conducted on a HP BL685C G7. All tools had up to eight cores with 2.0GHz and 8GB of memory available, but only the Java garbage collection of Prism and Epmc used more than one core. The timeout was 1800 seconds.

Benchmarks

All benchmarks from the Prism benchmark suite for DTMCs, MDPs and CTMCs were used.
