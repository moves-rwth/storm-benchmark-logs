Setup

The benchmarks were run on 4 cores of an Intel Xeon Platinum 8160 Processor with 12GB of memory available. The timeout was 1800 seconds. Storm was considered in Version 1.4.1

All benchmarks from QComp 2019 were considered. 4 PTA Benchmarks that were not compatible with Storm were not considered.

Benchmark Results from January 2017

Setup

The benchmarks were conducted on a HP BL685C G7. All tools had up to eight cores with 2.0GHz and 8GB of memory available, but only the Java garbage collection of Prism and Epmc used more than one core. The timeout was 1800 seconds.

Benchmarks

All benchmarks from the Prism benchmark suite for DTMCs, MDPs and CTMCs were used.
